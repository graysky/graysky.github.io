# graysky.org

This is the code behind my website [graysky.org](http://graysky.org). It uses Jekyll to generate static html.

## Run Locally

./bin/serve

Or like: bundle exec jekyll serve --drafts --watch --limit_posts 50

## Deploy to GitHub Pages

Just push to master!
